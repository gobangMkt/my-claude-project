# my-claude-project

개인 프로젝트들을 통합 관리하는 저장소입니다.

## 🚀 빠른 시작

```bash
# 첫 번째 노트북
git clone https://github.com/gobangMkt/my-claude-project
cd my-claude-project

# 다음번부터
git pull
cd 프로젝트/RoutineLog-  # 원하는 프로젝트로 이동
```

## 📁 프로젝트 구조

```
my-claude-project/
├── 프로젝트/
│   ├── RoutineLog-/              ✅ React + TypeScript 루틴 기록 앱
│   ├── roomrecommand/            🔄 TypeScript 방 추천 서비스
│   ├── youth-housing-diagnosis/  🔄 JavaScript 청년주택 진단
│   ├── blog_request/             🔄 HTML 블로그 요청
│   └── youthissue/               🔄 TypeScript 청년 이슈
├── CLAUDE.md                     📋 통합 개발 가이드
└── README.md                     📖 이 파일
```

## 📖 문서

- **[CLAUDE.md](./CLAUDE.md)**: 모든 프로젝트가 따라야 할 규칙 및 설정
